## Dependencies

* NodeJS
* MongoDB
* PhantomJS
* AWS bucket

## Installation
* `sudo npm install -g gulp migrate`
* `npm install`
* `migrate up`

## Serve Locally
* `gulp`

## Create project
* `gulp create-project --name="your project name"`
